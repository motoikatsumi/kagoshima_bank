# 鹿児島銀行レポートシステム — Windows セットアップマニュアル

## 1. システム概要

鹿児島銀行のネットバンキングにログインし、5店舗（本店・宇宿支店・伊敷支店・国分支店・寿支店）の取引履歴を取得して、Slackの各チャンネルに自動投稿するシステムです。

### 動作イメージ

- **アイコンダブルクリック** → アプリ起動（タスクトレイに常駐）
- **スケジュール実行** → 8:50 / 11:50 / 14:50 / 18:20 に自動で処理
- **手動実行** → トレイアイコン右クリック →「今すぐ実行」
- **Windows起動時** → 自動的にアプリが立ち上がる
- **スリープ解除** → PCがスリープ中でも自動で起こして実行（タスクスケジューラ連携）

---

## 2. 前提条件

| 項目 | 要件 |
|------|------|
| OS | Windows 10 / 11（64bit） |

| Python | **3.9 以上**（3.11 推奨） |
| ブラウザ | **Google Chrome**（最新版） |
| ネットワーク | インターネット接続（銀行サイト・Slack API） |

## 3. セットアップ手順

### 3-1. ファイルの配置

`windows` フォルダの中身を、Windows PCの任意の場所にコピーします。

**推奨配置先:**
```
C:\KaginReport\
  ├── kagin_app.py                  ← メインアプリ（トレイ常駐）
  ├── kagin_slack_report_win.py     ← 銀行処理スクリプト
  ├── .env                          ← Slackトークン設定（Git管理外）
  ├── requirements_win.txt          ← 依存パッケージ一覧
  ├── build.bat                     ← exeビルド用
  ├── setup.bat                     ← 一括セットアップ
  └── generate_icon.py              ← アイコン生成用
```

---

### 3-2. setup.bat を実行（全自動セットアップ）

**`setup.bat` をダブルクリックするだけで以下が全て自動で行われます：**

```
C:\KaginReport\setup.bat をダブルクリック
```

`setup.bat` が実行する内容：

| ステップ | 内容 |
|----------|------|
| **[1/6]** | **Python 自動インストール** — 未インストールなら `winget` で自動導入 |
| **[2/6]** | **Chrome 確認** — 未インストールなら自動 or 手動インストールを案内 |
| **[3/6]** | **依存パッケージ** — `pip install -r requirements_win.txt` を自動実行 |
| **[4/6]** | **.env 設定** — Slackトークンの入力（未設定の場合） |
| **[5/6]** | **自動起動設定** — Windows起動時の自動起動ON/OFF |
| **[6/6]** | **スリープ解除設定** — タスクスケジューラでスリープ解除ON/OFF |

その後、パスワード設定の案内が表示されます。

> **💡 `winget` について:** Windows 10（バージョン1709以降）/ Windows 11 に標準搭載のパッケージマネージャーです。`winget` が使えない古い環境では、ブラウザが自動で開き手動インストールを案内します。

#### 手動でインストールする場合（参考）

<details>
<summary>Python を手動でインストールする場合</summary>

1. https://www.python.org/downloads/ からPython 3.11以上をダウンロード
2. インストーラーを実行
3. **⚠️「Add Python to PATH」に必ずチェックを入れる**
4. 「Install Now」をクリック

確認: コマンドプロンプトで `python --version` を実行

</details>

<details>
<summary>Chrome を手動でインストールする場合</summary>

1. https://www.google.com/chrome/ からダウンロード・インストール
2. 一度起動して初期設定を完了させておく

</details>

### 3-3. exe ファイルの作成（任意）

コマンドプロンプトで以下を実行するか、`build.bat` をダブルクリック：
```
cd C:\KaginReport
build.bat
```

ビルドが成功すると `dist\KaginReport.exe` が生成されます。

**生成されたファイルの配置:**
```
C:\KaginReport\
  ├── KaginReport.exe    ← dist\ からここにコピー
  └── （その他ファイル）
```

> **💡 ヒント:** exe を使わずに Python で直接実行することも可能です（後述）。

## 4. 初回起動

### 4-1. アプリの起動

`KaginReport.exe` をダブルクリックします。

または Python から直接実行：
```
cd C:\KaginReport
python kagin_app.py
```

### 4-2. パスワードの設定

初回起動時に自動的にパスワード入力ダイアログが表示されます。

1. 鹿児島銀行のログインパスワードを入力
2. 「OK」をクリック
3. 「パスワードを保存しました」と表示されれば完了

> パスワードは **Windows資格情報マネージャー** に安全に保存されます。
> 平文でファイルに保存されることはありません。

### 4-3. 動作確認

- タスクトレイ（画面右下の時計の横）にアイコンが表示されていることを確認
- アイコンを **右クリック** → **「▶ 今すぐ実行」** で手動テスト
- 処理中はアイコンが黄色に変わります
- 完了すると通知が表示され、アイコンが元の色に戻ります

## 5. Windows 起動時の自動起動設定

### 方法A: アプリから設定（推奨）

1. タスクトレイのアイコンを右クリック
2. **「Windows起動時に自動起動」** をクリック
3. ✅ マークが付けばON

### 方法B: setup.bat から設定

1. `setup.bat` をダブルクリック
2. 画面の指示に従って `Y` を入力

### 方法C: 手動でスタートアップに登録

1. `Win + R` → `shell:startup` → Enter
2. 開いたフォルダに `KaginReport.exe` のショートカットを作成

---

## 5-b. スリープ解除設定

PCがスリープ・休止状態でも、スケジュール時刻になると自動的にPCが起動して処理を実行する機能です。
Windowsのタスクスケジューラに「スリープ解除」タスクを登録します。

### 方法A: アプリから設定（推奨）

1. タスクトレイのアイコンを右クリック
2. **「スリープ解除で自動実行」** をクリック
3. ✅ マークが付けばON

### 方法B: setup.bat から設定

1. `setup.bat` をダブルクリック
2. ステップ [6/6] で `Y` を入力

> **⚠️ 注意:**
> - 電源プランの設定で「スリープ解除タイマーを許可」が有効になっている必要があります
>   - `コントロールパネル → 電源オプション → プラン設定の変更 → 詳細な電源設定`
>   - `スリープ → スリープ解除タイマーの許可 → 有効`
> - ノートPCで蓋を閉じている場合、BIOS/UEFI設定によってはスリープ解除できないことがあります

## 6. 操作方法

### タスクトレイメニュー

アイコンを右クリックすると以下のメニューが表示されます：

| メニュー項目 | 説明 |
|---|---|
| 次回: XX:XX | 次のスケジュール実行時刻 |
| **▶ 今すぐ実行** | 手動で即時に処理を実行 |
| **🔑 パスワード設定** | 銀行パスワードの変更・再入力 |
| **Windows起動時に自動起動** | ON/OFF切替（✅ = ON） |
| **スリープ解除で自動実行** | ON/OFF切替（✅ = ON）— PCがスリープ中でも自動起動 |
| **📄 ログを開く** | ログファイルをメモ帳で表示 |
| **終了** | アプリを終了 |

### スケジュール実行時刻

| 時刻 | 備考 |
|------|------|
| 08:50 | 朝一 |
| 11:50 | 昼前 |
| 14:50 | 午後 |
| 18:20 | 夕方 |

- PCがスリープ・休止状態から復帰した場合、予定時刻から **10分以内** なら遅延実行されます
- 既に実行済みの時刻は再実行されません
- **スリープ解除機能を有効にすると**、タスクスケジューラがスケジュール時刻にPCを自動的にスリープから復帰させます

### アイコンの色

| 色 | 状態 |
|----|------|
| 紺色 | 待機中（正常） |
| 黄色 | 処理実行中 |
| 赤色 | エラー発生 |

## 7. トラブルシューティング

### 「パスワードが未設定です」と表示される

→ トレイアイコン右クリック →「🔑 パスワード設定」からパスワードを入力してください。

### 処理が失敗する・アイコンが赤くなる

1. トレイアイコン右クリック →「📄 ログを開く」でログを確認
2. よくある原因：
   - インターネット接続が切れている
   - 銀行サイトがメンテナンス中
   - パスワードが変更された
   - Chromeが古いバージョン

### Chrome関連のエラー

ChromeDriverのバージョン不一致の場合：
```
pip install --upgrade selenium
```
Selenium 4.6以降は ChromeDriver を自動管理するため、通常は手動更新不要です。

### アプリが起動しない

- Python がPATHに通っているか確認：`python --version`
- 依存パッケージがインストールされているか確認：`pip list`
- ログファイル `kagin_app.log` を確認

### exeのビルドが失敗する

```
pip install --upgrade pyinstaller
```
を実行してから再度 `build.bat` を実行してください。

## 9. macOS版との対応表

| 機能 | macOS | Windows |
|------|-------|---------|
| パスワード保管 | Keychain (`security`コマンド) | Windows資格情報マネージャー (`keyring`) |
| スリープ防止 | `caffeinate` | `SetThreadExecutionState` API |
| 定期実行 | launchd + pmset | アプリ内蔵スケジューラ + タスクスケジューラ（スリープ解除） |
| 自動起動 | LaunchAgents plist | レジストリ `HKCU\...\Run` |
| スクリプト実行 | `.sh` シェルスクリプト | アプリに統合（exe 1つ） |

## 10. アンインストール

1. タスクトレイからアプリを「終了」
2. 自動起動が有効な場合はOFFにする（トレイメニューから）
3. スリープ解除が有効な場合はOFFにする（トレイメニューから）  
   または手動で削除: `schtasks /Delete /TN KaginReport_Wake_0850 /F` 等
4. `C:\KaginReport` フォルダを削除
5. （任意）Windows資格情報マネージャーから `kagin_bank` を削除
   - `Win + R` → `control /name Microsoft.CredentialManager` → Enter
   - 「Windows資格情報」→「kagin_bank」→「削除」
